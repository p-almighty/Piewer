# Piewer

**Windows 向け ローカル漫画ビュワー**

登録数無制限・本棚スクロール位置の記憶・PDF対応など、<br>
快適に漫画を読むための機能を詰め込んだ無料ビュワーです。

## 📥 ダウンロード

[**Releases ページ**](../../releases/latest) から入手できます。2種類を用意しています。

| 種類 | ファイル | おすすめの方 |
|------|---------|------------|
| **インストール版** | `Piewer_Setup.exe` | PCに入れてスタートメニューから使いたい方 |
| **非インストール版（ポータブル）** | `Piewer.exe` | インストールせず使いたい／持ち運びたい方 |

> Pythonのインストールは不要です。ダウンロードしてすぐ使えます。

---

## ✨ 主な機能

- 📚 **複数の本棚** — ジャンル別などで本棚を分けて管理
- 🚀 **登録数無制限** — 数千冊でも快適な仮想スクロール
- 📖 **見開き表示** — 右綴じ/左綴じ、奇数/偶数始まりに対応
- 🔍 **ズーム & パン** — マウスホイールで最大8倍
- 🖼️ **ページサムネイル** — 一覧から好きなページへ一瞬でジャンプ
- 📂 **多彩な形式に対応** — ZIP / CBZ / RAR / CBR / **PDF** / 画像フォルダ
- 💾 **続きから再開** — 最後に読んだページを記憶
- 🎯 **まとめて管理** — Shiftクリックで範囲選択して一括削除
- 🌙 **ダークテーマ** — 目に優しい配色

---

## 📂 対応ファイル形式

`ZIP` / `CBZ` / `RAR` / `CBR` / `PDF` / 画像フォルダ
（内包画像: JPG / PNG / GIF / BMP / WEBP）

## ⌨️ 操作方法

| 操作 | 動作 |
|------|------|
| 画面 左/右クリック | ページを戻る / 進む |
| 画面 中央クリック・右クリック | メニュー(HUD)表示 |
| ドラッグ | ページめくり |
| ホイール | ズーム |
| ← → / A D | ページ移動 |
| F11 | 全画面切り替え |
| Esc | 全画面解除 / 本棚に戻る |

詳しい使い方は同梱の `readme.txt` をご覧ください。
