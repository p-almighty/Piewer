============================================================
  Piewer （ピューワー）  -  マンガビュワー
  Version 1.81
============================================================

Windows 11 向けのローカル漫画ビュワーアプリです。
本棚管理・縦読み対応・日英対応など、快適に漫画を読むための
機能を詰め込んでいます。

【完全無料・オープンソース】
  ・Piewer は完全無料です。登録数の制限なく、すべての機能を
    無料でお使いいただけます。
  ・ソースコードは MIT ライセンスで公開しています。
    https://github.com/p-almighty/Piewer
  ・気に入っていただけたら、開発の支援（寄付・任意）にも
    対応しています（「⚙ 設定 → 開発を支援」または Ko-fi）。
    https://ko-fi.com/p_almighty

【言語 / Language】
  ・日本語 / English に対応。「⚙ 設定 → 言語 / Language」で
    いつでも切り替えられます（即時反映）。


------------------------------------------------------------
■ 配布形態（2種類）
------------------------------------------------------------

  ・インストール版    : Piewer_Setup.exe
        実行するとPCにインストールされ、スタートメニュー
        （プログラム一覧）に登録されます。アンインストールは
        「アプリと機能」から行えます。

  ・非インストール版  : Piewer.exe（ポータブル版）
        コピーするだけで起動できます。USBメモリ等での
        持ち運びにも使えます。


------------------------------------------------------------
■ インストール / 起動方法
------------------------------------------------------------

1. 「Piewer.exe」をお好きなフォルダに置いてください。
   （インストール作業は不要です。ダブルクリックで起動します）

2. 初回起動時、以下のフォルダが自動的に作成されます。
       C:\Users\(ユーザー名)\.manga_viewer\
   ここに本棚データ・設定・表紙画像が保存されます。
   アプリ本体を移動・削除してもデータは残ります。

※ exe単体で動作します。Pythonのインストールは不要です。


------------------------------------------------------------
■ 対応ファイル形式
------------------------------------------------------------

  ・ZIP / CBZ            （画像をまとめた書庫）
  ・EPUB / KEPUB         （画像ベースの電子書籍・漫画。Koboの.kepub.epubも可）
  ・RAR / CBR            （※環境により unrar が必要）
  ・PDF                  （各ページを画像として表示）
  ・画像フォルダ          （フォルダ内の画像を1冊として表示）
  ・内包画像形式: JPG / JPEG / PNG / GIF / BMP / WEBP
                （動くGIF / WebP / APNG はページ内で自動再生）

  ※ Kindleの KFX には非対応です。ご自身が所有する本は、Calibre
    （無料・kfx Input プラグイン）で EPUB / CBZ に変換すると開けます。


------------------------------------------------------------
■ 基本的な使い方
------------------------------------------------------------

【本棚を選ぶ】
  起動すると本棚一覧が表示されます。本棚カードをクリックして
  中に入ります。「新しい本棚」で追加・名前変更・削除ができます。
  一覧の左側には ★お気に入り・🕒最近読んだ本 の専用棚があります。

【漫画を追加する】
  本棚の中で「＋ ファイル」ボタンから選択するか、ウィンドウへ
  ファイル/フォルダを ドラッグ＆ドロップ して追加できます。

【漫画を読む】
  本のカードをクリックすると開きます（全画面はF11やメニューから）。
  既読の本は既定で「続きから」開きます（⚙設定→「本を開いたとき」で
  毎回確認 / 最初から に変更できます）。

【登録せずフォルダから開く】
  本棚一覧の「📁 フォルダから開く」で、PCのフォルダをたどって
  圧縮ファイルや画像フォルダをその場で開けます（本棚への登録は不要）。


------------------------------------------------------------
■ リーダー操作一覧（横読み）
------------------------------------------------------------

  画面 左/右クリック     … ページを戻る / 進む
  画面 中央/右クリック   … メニュー(HUD)の表示/非表示
  ドラッグ（スワイプ）   … ページめくり（タッチ対応）
  マウスホイール         … ズーム（マウス位置を中心に拡大／幅フィット時は縦スクロール）
  上下ドラッグ           … 無段階ズーム（マンガミーヤ式・ポインタ中心。⚙設定でON/OFF）
  ズーム中ドラッグ       … 表示位置の移動（パン）
  ズーム中の中央ダブルクリック … 等倍に戻す
  🗂 目次                … 全ページのサムネイル一覧から選んでジャンプ

  ← → ↑ ↓ / A D W S    … ページ移動
  [  /  ]               … 前 / 次のしおりへジャンプ
  Home / End            … 最初 / 最後のページへ
  F11                   … 全画面の切り替え
  Esc                   … 本棚に戻る
  マウス「戻る」ボタン    … 本棚に戻る
  マウス「進む」ボタン    … 直前に開いた本を再開

  ※ キー割り当ては「?」ヘルプ →「ショートカット設定」で変更できます。


------------------------------------------------------------
■ 縦読み（Webtoon）モード
------------------------------------------------------------

  メニューの「縦読み」ボタンで、ページを縦に連続表示します。

  S・↓ / A・←          … 次のページ
  W・↑ / D・→          … 前のページ
  画面 左端クリック       … 次のページ
  画面 右端クリック       … 前のページ
  画面 中央クリック       … メニュー(HUD)の表示/非表示
  上下ドラッグ           … ページめくり（指/マウスに追従）
  マウスホイール         … ページ送り
  タッチ                 … 慣性スクロール


------------------------------------------------------------
■ メニュー(HUD)の機能
------------------------------------------------------------

  上部ツールバー:
    ・本棚へ戻る / 最後・次へ・前へ・最初
    ・ページ番号を入力して任意のページへジャンプ
    ・🔖 しおり（現在ページに目印）/ 前後のしおりへ / 一覧
    ・右→左 / 見開き / 奇数・偶数始まり / 幅フィット / 縦読み
    ・全画面 切り替え
    ※ 右綴じ・見開き・幅・縦読みなどの表示設定は本ごとに記憶されます。

  下部サムネイルストリップ:
    ・全ページのサムネイルを一覧表示（現在ページは緑の枠）
    ・しおりページには金色の目印
    ・クリックでそのページへ移動／ホイールで横スクロール
    ・右側が1ページ目（右綴じ漫画準拠）


------------------------------------------------------------
■ 本棚ビューの機能
------------------------------------------------------------

  ・⭐お気に入り棚 / 🕒最近読んだ本棚（専用の本棚で素早くアクセス）
  ・お気に入り … カード右上の★をクリックで登録/解除
  ・タグ付け … カードを右クリック →「タグを編集」。既存タグはクリックで
        追加でき、カード左上のタグアイコンで確認できます
  ・絞り込み … お気に入り / タグで本棚を絞り込み（🏷 絞り込みボタン）。
        タグは種類（作者/サークル/原作/イベント）ごとに折りたたみ表示・検索でき、
        ひらがな/カタカナ・全角/半角を区別せず探せます
  ・自動タグ付け（実験的）… ファイル名から作者・サークル・原作・イベントを
        抽出してタグ付け（⚙設定→タグの管理／追加時に自動付与のON切替も可）
  ・検索 … タイトル検索（ひらがな/カタカナ・全角/半角を区別せず候補も表示）。
        「全棚」で全本棚を横断検索
  ・並び替え … 登録順 / ファイル名 / 最近読んだ順 / 進捗順
  ・別の本棚へ移動 … 右クリックメニュー、または選択削除モードで一括移動
  ・サイズ切り替え（小 / 中 / 大）
  ・選択削除モード（クリックで選択、Shiftで範囲選択、一括削除/移動）
  ・未読は黒、既読は緑のオーバーレイ＋読書進捗バー
  ・数千冊でも快適な仮想スクロール（高リフレッシュレート対応）
  ・マウスのドラッグ / タッチのスワイプでスクロール（慣性付き）
  ・ホイールクリックで自動スクロール

  「⚙ 設定」ボタンから:
    ・本棚の名前変更
    ・サムネイル（表紙）の再生成
    ・カバー画像の保存先の変更
    ・本棚を開いたときのスクロール位置（前回の位置 / 一番上）
    ・タグの一括管理（名前変更・削除）
    ・バックアップの保存 / 復元


------------------------------------------------------------
■ データの保存場所
------------------------------------------------------------

  C:\Users\(ユーザー名)\.manga_viewer\
    ├── library.json     … 本棚・書籍情報
    ├── settings.json    … 設定（サムネイルサイズ・ショートカット等）
    └── covers\          … 表紙画像のキャッシュ（保存先は変更可能）

  ※ 表紙キャッシュの保存先は「⚙ 設定」から変更できます。
  ※ 登録するのはファイルの「場所(パス)」です。
     元のファイルを移動・削除すると開けなくなります。
  ※ 「⚙ 設定」→ バックアップで library/設定を1ファイルに保存できます。

■ 補足・既知の制限
------------------------------------------------------------

  ・RAR/CBR を開くには環境によって unrar が必要な場合があります。
  ・PDFはページを画像化して表示します。ページ数が多い場合は
    初回サムネイル生成に少し時間がかかることがあります。
  ・PyInstaller製のため、一部のセキュリティソフトで誤検知される
    ことがあります（公式 GitHub で配布している正規のファイルです）。
    ・SmartScreen の警告は［詳細情報］→［実行］で起動できます。
    ・ソフトが削除/隔離した場合は、隔離から復元し Piewer.exe を
      例外（許可リスト）に追加してください。
    ・本ソフトは完全オープンソース（MIT）です。不安な場合は
      ソースから自分でビルドして実行できます。
    ・誤検知は Microsoft の報告フォームから報告にご協力いただけると
      改善に役立ちます: https://www.microsoft.com/en-us/wdsi/filesubmission
