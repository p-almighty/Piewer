============================================================
  Piewer （ピューワー）  -  マンガビュワー
  Version 1.0
============================================================

Windows 11 向けのローカル漫画ビュワーアプリです。
登録数無制限・本棚スクロール位置の記憶など、市販ビュワーの
不満点を解消するために自作されました。


------------------------------------------------------------
■ 配布形態（2種類）
------------------------------------------------------------

  ・インストール版    : Piewer_Setup.exe
        実行するとPCにインストールされ、スタートメニュー
        （プログラム一覧）に登録されます。アンインストールは
        「アプリと機能」から行えます。

  ・非インストール版  : Piewer.exe（ポータブル版）
        コピーするだけで起動できます。USBメモリ等での
        持ち運びにも使えます。


------------------------------------------------------------
■ インストール / 起動方法
------------------------------------------------------------

1. 「Piewer.exe」をお好きなフォルダに置いてください。
   （インストール作業は不要です。ダブルクリックで起動します）

2. 初回起動時、以下のフォルダが自動的に作成されます。
       C:\Users\(ユーザー名)\.manga_viewer\
   ここに本棚データ・設定・表紙画像が保存されます。
   アプリ本体を移動・削除してもデータは残ります。

※ exe単体で動作します。Pythonのインストールは不要です。


------------------------------------------------------------
■ 対応ファイル形式
------------------------------------------------------------

  ・ZIP / CBZ            （画像をまとめた書庫）
  ・RAR / CBR            （※環境により unrar が必要）
  ・PDF                  （各ページを画像として表示）
  ・画像フォルダ          （フォルダ内の画像を1冊として表示）
  ・内包画像形式: JPG / JPEG / PNG / GIF / BMP / WEBP


------------------------------------------------------------
■ 基本的な使い方
------------------------------------------------------------

【本棚を選ぶ】
  起動すると本棚一覧が表示されます。
  本棚カードをクリックして中に入ります。
  「新しい本棚」で本棚を追加・名前変更・削除できます。

【漫画を追加する】
  本棚の中で「＋ ファイル」ボタンから漫画ファイルを
  選択して追加します（複数同時選択可）。
  追加した本は本棚の左上に表示されます。

【漫画を読む】
  本のカードをクリックすると全画面で開きます。
  既読の本は「最初から / 続きから」を選べます。


------------------------------------------------------------
■ リーダー操作一覧
------------------------------------------------------------

  画面 左クリック       … ページを戻る/進む（左1/3）
  画面 右クリック       … メニュー(HUD)の表示/非表示
  画面 中央クリック     … メニュー(HUD)の表示/非表示
  ドラッグ（スワイプ）   … ページめくり
  マウスホイール         … ズーム（カーソル位置中心・最大8倍）
  ズーム中ドラッグ       … 表示位置の移動（パン）

  ← → ↑ ↓ / A D W S    … ページ移動
  Home / End            … 最初/最後のページへ
  F11                   … 全画面の切り替え
  Esc                   … 全画面解除 / 本棚に戻る
  マウス「戻る」ボタン    … 本棚に戻る
  マウス「進む」ボタン    … 直前に開いた本を再開


------------------------------------------------------------
■ メニュー(HUD)の機能
------------------------------------------------------------

  上部ツールバー:
    ・本棚へ戻る
    ・最後 / 次へ / 前へ / 最初 ボタン
    ・ページ番号を入力して任意のページへジャンプ
    ・右→左 / 見開き / 奇数始まり / 偶数始まり 切り替え
    ・全画面 切り替え

  下部サムネイルストリップ:
    ・全ページのサムネイルを一覧表示
    ・現在表示中のページを緑の枠で表示
    ・ホイールで横スクロール
    ・サムネイルをクリックでそのページへ一瞬で移動
    ・右側が1ページ目、左へ進むほど後のページ（右綴じ漫画準拠）


------------------------------------------------------------
■ 本棚ビューの機能
------------------------------------------------------------

  ・サムネイル(表紙)の自動生成
  ・サイズ切り替え（小 / 中 / 大）
  ・並び替え（登録順 / ファイル名順）
  ・タイトル検索
  ・選択削除モード（複数まとめて削除）
        - 本をクリックして1冊ずつ選択/解除
        - Shiftキーを押しながらクリックすると、直前に
          選んだ本からその本までを一括で範囲選択できます
        - 全選択 / 全解除 / 選択した本を削除
  ・サムネイル再生成（表紙を作り直す）
  ・未読は黒、既読は緑のオーバーレイ表示＋読書進捗バー
  ・数千冊でも快適な仮想スクロール
  ・ホイールクリックで自動スクロール（ブラウザ風）


------------------------------------------------------------
■ データの保存場所
------------------------------------------------------------

  C:\Users\(ユーザー名)\.manga_viewer\
    ├── library.json     … 本棚・書籍情報
    ├── settings.json    … 設定（サムネイルサイズ等）
    └── covers\          … 表紙画像のキャッシュ

  ※ 登録するのはファイルの「場所(パス)」です。
     元のファイルを移動・削除すると開けなくなります。


------------------------------------------------------------
■ 動作環境
------------------------------------------------------------

  ・Windows 11（Windows 10でも動作可）
  ・追加のインストール不要（exe単体動作）


------------------------------------------------------------
■ 補足・既知の制限
------------------------------------------------------------

  ・RAR/CBR を開くには環境によって unrar が必要な場合があります。
  ・PDFはページを画像化して表示します。ページ数が多い場合は
    初回サムネイル生成に少し時間がかかることがあります。
  ・登録できる冊数に上限はありません。
